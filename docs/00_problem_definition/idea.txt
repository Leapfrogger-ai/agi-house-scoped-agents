Each task declaration includes a machine-readable intent manifest (what tools, what scope, what time budget)

The agent runs inside a Daytona ephemeral sandbox — isolated compute, scoped filesystem, auto-destroyed on task completiondaytonayoutube

A lightweight conformance monitor watches tool calls against the declared intent and halts + logs on divergence (the "intent-conformance monitor" idea from the blog)1password

Every action emits a structured audit event binding: agent ID → human delegator → credential used → intent → outcome
