# Vincent

> Vincent is an open framework by Lit Protocol for building programmable, user-controlled agents and vaults. Developers ship agents that automate operations across blockchains, DeFi, and traditional apps, while users keep full custody of their assets and data through policy-enforced delegation.

Vincent is built on Lit Protocol's decentralized key management (Programmable Key Pairs and Lit Actions). The core model has four parts: **Agent Wallets** (non-custodial accounts that delegate signing authority), **Abilities** (immutable functions defining what an app can do), **Policies** (programmable guardrails like spending limits and rate limits that gate every ability execution), and **Apps** (policy-governed automation packages a user approves on the Connect Page). Approvals and authorizations are tracked in an on-chain App Registry. Doc links below point at the source `.mdx` files in this repository; the `## Optional` section may be skipped when context is limited.
